Version 2.0 · Made by Incredible Projects / Henk Jekel

What is ERPIE?

ERPIE (Expected Resale Price Intelligence Engine) calculates the expected resale value of refurbished IT assets for the B2B market. It's designed for IT Asset Disposition (ITAD) companies.

How to use

  1. Select device details — Brand, model, CPU, RAM, SSD, etc.
  2. Check live prices — Use the search buttons to validate prices on B2B and B2C marketplaces.
  3. Get AI Analysis — Click the purple button for an AI-powered price estimate with confidence score.

Price Sources

  • B2B (green) — IT Gigant, Troostwijk Auctions — wholesale prices
  • B2C (blue) — Back Market, Refurbed, Amazon — consumer reference prices
  • Apple (purple) — RefurbMe — Apple refurbished price comparison

Keyboard Warnings

  • 🟢 QWERTY EU/UK/US — Full resale value
  • 🟡 QWERTZ DE, Nordic — Limited market (-20%)
  • 🔴 QWERTZ CH (Swiss) — No resale value (€0)

Serial Number Check

Enter a serial number to check warranty status and device specs on the manufacturer's support site.
